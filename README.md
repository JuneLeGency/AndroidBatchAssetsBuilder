AndroidBatchAssetsBuilder
====================
android 图标多屏幕适配制作十分麻烦
这个项目就是为了解决该问题
**使用网址[打开](http://android.letvr.com/)**

现在支持的图标

-批量处理图标
-启动图标
-Action bar 和 tab 中的图标
-通知栏图标
-通用的普通图标
